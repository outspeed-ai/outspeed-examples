# Setup

```
npm i # install the packages
npm run dev # run
```


# Key points

1. First we need to setup tailwind correctly.
  1. Add `darkMode`
  2. Add all the colors.
  3. Make sure background and foreground uses hsl.
  4. Add `@outspeed/react/**/*.{js,mjs}` in content.
2. In layout.tsx, add `dark` class in html.
